# bridge-soil-moisture 27/04/2026
🌱 Project Description

The Soil Moisture Monitoring System is an Arduino-based project designed to measure the water content in soil and provide a simple visual indication of its condition. The system uses a soil moisture sensor to continuously detect moisture levels and sends this data to the Arduino Uno, which processes the information and determines whether the soil is dry or sufficiently wet.

Based on the measured values, the system activates two LEDs: a red LED to indicate dry soil that requires watering, and a green LED to indicate adequate moisture. This real-time feedback helps users make better irrigation decisions and avoid overwatering or underwatering plants.

The project is low-cost, easy to build, and suitable for beginners learning embedded systems and sensor integration. It also demonstrates a basic concept of smart agriculture, where technology is used to improve efficiency in plant care. With further development, the system can be expanded to automatically control water pumps or be integrated into IoT-based smart farming solutions.
